Comparisons

VPN providers

TL;DR Don't use one, instead use TOR or I2P, if thats not an option buy IVPN, OVPN or Mullvad VPN preferably via Monero or cash

Web browsers

TL;DR If you use Firefox switch to GNU Icecat, if you use Chrome switch to Ungoogled Chromium, if you want something minimal use qutebrowser

Instant messaging services

TL;DR If you want something modern with multimedia, screenshare and VC use Matrix with the nheko client

Search engines

TL;DR Avoid closed source search engines, use LibreX, SearXNG or Whoogle

Git services

TL;DR If you want to self host use cgit, otherwise use sourcehut or Codeberg
